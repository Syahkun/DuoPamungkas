# DuoPamungkas
Project REST API menampilkan menu makanan sehat per hari berdasarkan daily calorie intake untuk mengurangi, menambah, atau mempertahankan berat badan (sesuai kebutuhan) berdasarkan perhitungan BMI dan BMR

# Keterangan
Kedua API yang digunakan bersifat FREEPREMIUM sehingga terdapat KUOTA REQUEST HARIAN atau BULANAN yang harus diperhatikan!

# Flow
Branch tiap anggota > Development > Master
